# Keymaps

This file is maintained by code-aware audit of the Neovim config in this repo. It is not generated by a static parser.

Notes:
- `<leader>` is `<Space>`.
- `<localleader>` is also `<Space>`.
- This document separates keymaps defined in this repo from keymaps provided by dependency/downstream plugins and from trigger-only/plugin-default entries.
- When completeness matters, dependency plugins should be inspected and previously audited plugin versions should be reused where possible.
- Related plugins or features that are always disabled should be called out explicitly.

## Explicitly disabled plugins or features

- `zk.nvim` note LSP auto-attach is disabled via `lua/plugins/zk.lua` â†’ `lsp.auto_attach.enabled = false`.
  - The ZK keymaps below are still active.
  - Only the automatic ZK LSP attachment behavior is disabled.
